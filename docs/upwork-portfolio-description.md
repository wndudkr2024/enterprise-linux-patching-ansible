# Upwork Portfolio Description

## Enterprise Linux Patch Automation with Ansible

Built a reusable Ansible automation framework for patching RHEL-compatible Linux servers across dev and production-style environments.

The project includes role-based automation, hierarchical inventories, group variables, pre-check and post-check validation, reboot handling, package exclusion controls, and patch execution modes for both security-only and full updates.

This portfolio demonstrates my ability to design maintainable infrastructure automation for enterprise Linux environments using Ansible best practices.
